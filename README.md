# pairwise-filtering-sorting
**Описание задания:**  
Проведено оптимизированное тестирование функций фильтрации и сортировки интернет-магазина с применением техники "Попарное тестирование".

**Цель:**  
Сократить количество проверок при тестировании комбинаций параметров, сохранив покрытие критических сценариев.

**Что было сделано:**  
1. Определены параметры для фильтрации:
   - Категория товара  
   - Производитель  
   - Бесплатная доставка  
2. Определены параметры для сортировки:
   - Сортировка по имени 
   - Сортировка по цене 
3. Создана таблица исходных данных  
4. С помощью инструмента  **teremok** сгенерированы тестовые комбинации по алгоритму Pairwise
